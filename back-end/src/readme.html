ℹ️ About

This document provides a brief description of api methods.

Methods are divided into 🔓public and 🔒private.

Private methods are only available with user credentials passed.

🔓 Public methods

Create account

POST:/auth/signup
Request body:
              {
                    "login":"1",
                    "pwd":"pwd",
                    "name":"mikola"
              }

Responce:
{
    "statusCode": 200,
    "data": {
        "hash": "636b73f6-e457-496c-be46-7875047e12b5",
        "name": "mikola",
        "status": true,
        "date": "2022-08-19T16:27:11.838Z"
                }
            }

Get auth data (login)

POST:/auth/signin
Request body:
                {
                    "login":"1",
                    "pwd":"pwd",
                }

Responce:
{
    "statusCode": 200,
    "data": {
        "sessionToken": "5d412dc2-786e-4b68-9276-63aab4460a9f",
        "loginMessage": "sussess",
        "name": "mikola",
        "userToken": "9d6196f2-e667-4904-bd40-1fb27024094c"
    }
}

Info.The sessionToken - session key is valid for 6 hours, without binding to the device.

Log out

GET:/auth/signout/{sessionToken}/{userToken}
Responce:
{
            "statusCode": 200,
            "data": {
                "isLogOut": true
                    }
                }
